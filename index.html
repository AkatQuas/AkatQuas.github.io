Wander in webpack

Posted on 2020-12-06 In frontend , javascript , webpack

Setup the project

git clone https://github.com/webpack/webpack.git && cd webpack

# maybe switch to some branch/tag

# manually
npm install -g yarn
yarn
yarn link
yarn link webpack

# automatically
npm run setup
# or
yarn setup

At webpack, yarn is preferred.

If you already have yarn installed, do: yarn setup. This will complete all required steps.

If not, do: npm run setup, the setup will also install yarn for you.

And you are good to go.

Useful tips

go tool compile -pack <source.go>

Structure and Interpretation of Computer Programs is the most useful (magic) book when you struggle in the computer science / programms / languages.

It teaches you to think the concepts in programming, rather just executing codes.

Fullstack Starter Project Boilerplate

Posted on 2020-04-01 In frontend

项目做的多了，所以就抽离出来一个全栈项目模板，前端使用 Angualr ( UI 库 使用 Ng-Zorro ) ，后端使用 Nest 。Project on GitHub.

有一些好用的脚本来帮助项目的起步、开发、部署。

Foundation

This boilerplate is build on Angular, Ng-Zorro-Antd and Nest.

Setup

setup_all.sh helps to set the development environment up, including:

  1. commitizen for git commit message

  2. nginx for universal gateway

  3. pm2 for develop convenience

  4. proxychains4 for proxy

./setup_all.sh

Development

Execute the dev_all.sh to develop.

./dev_all.sh

Build or Deployment

Use docker for build. Please check the DockerFile.
