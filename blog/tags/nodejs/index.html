设计一个 CLI 工具

June 11, 2022 · 7 min read

自从接手编辑器的 CLI 工具以来，我对原始的代码设计深恶痛绝。只是无奈重构、升级等迟迟无法提上日程，所以只能在这里做大梦。

常见工具​

在 Node 中，我接触过的 CLI 开发工具有

  • common-bin
  • commander
  • oclif

相比较而言，前两个 common-bin 和 commander 只能算是 CLI 的开发工具套件 (library) ，而 oclif 可以称得上 CLI 开发框架。

用 Web 生态去类比，前两者好比就是 React 和 Vue ，属于 Web UI library 。作为核心库，它们只提供了 MVVM 的功能，诸如 router 、store 、http 等都需要同生态下的其他 library 支持。

Read input from command line in node

October 28, 2019 · One min read
// read-from-terminal.js
process.stdin.resume();
process.stdin.setEncoding('ascii'); // 'utf-8' also supported

let _input = '';

process.stdin.on('data', function (data) {
  _input += data;
});

process.stdin.on('end', function () {
  process.stdout.write(_input);
});

usage example

you need to create a ReadStream using pipeline.

cat <file> | node read-from-terminal.js
