Propositional Logic 命题逻辑

命题逻辑是让可以对逻辑表达式的真假进行推理的数学模型。

命题：任何可以具有某一真值（真或加）的语句。

逻辑运算符：AND, OR, NOT, NAND, NOR。

NAND: 与非， p NAND q 等价于 NOT (p AND q)。 NOR: 或非， p NOR q 等价于 NOT (p OR q)。

基本规则：

  1. 当且仅当p和q都为真时，p AND q为真，否则为假。

  2. 当p为真，或者q为真，或者p和q都为真时，p OR q为真，否则为假。

  3. 当p为真时，NOT p为假，如果p为假时，NOT p为真。

运算符的优先级：NOT -> NAND -> NOR -> AND -> OR

布尔函数​

接受真值赋值作为参数，并返回 TRUE 或者 FALSE 的函数。

BOOLEAN foo( BOOLEAN p, BOOLEAN q)
{
    return p && (p||q);
}

很容易的就能把布尔函数表示为真值表。
