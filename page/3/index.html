Zero-width space

Posted on 2019-11-01

Insert zero-width space to cheat keywords detection without losing reading convenience.

function insertSpaces(string) {
  return Array.from(string).join('\u200b');
}

The zero-width space character is encoded in Unicode as U+200B (HTML entity &#8203;)

Read input from command line in node

Posted on 2019-10-28 In command line 
// read-from-terminal.js
process.stdin.resume();
process.stdin.setEncoding('ascii'); // 'utf-8' also supported

let _input = '';

process.stdin.on('data', function (data) {
  _input += data;
});

process.stdin.on('end', function () {
  process.stdout.write(_input);
});
